# Vuejs app with apollo

## Installation

`nvm use 18.20.4`

`yarn`

### Previously performed installations

`yarn add vuetify`

`yarn add graphql graphql-tag @apollo/client`

## Commands

`yarn dev` : compile and Hot-Reload for Development

`yarn build : Type-Check, Compile and Minify for Production`

`yarn test:unit`

`yarn test:e2e:dev`

`yarn lint`

### For CI

```sh
yarn build
yarn test:e2e
```
